# Java Feature Playground

本项目用于学习和实践 Java 8 之后引入的语言新特性，通过编写独立示例代码，加深对语法机制、使用方式以及实际应用场景的理解。

## 项目用途

该仓库作为个人 Java 语言特性学习实验库，用于系统掌握现代 Java 的核心语法能力，并作为长期技术积累与复习参考。

## 覆盖知识点

本项目计划涵盖以下 Java 新特性：

* Lambda 表达式
* Stream API
* Optional
* 新日期时间 API（LocalDateTime 等）
* Record 类型
* Switch 新语法
* instanceof 模式匹配
* Text Block 文本块
* Sealed Class 密封类

## 说明

本项目仅用于学习与实验，不用于生产环境。
